# Heart-Disease-Prediction-Using-Machine-Learning-
This project implements a Heart Disease Prediction system using Logistic Regression in Python. The model predicts whether a person is likely to have heart disease based on clinical parameters such as age, cholesterol level, resting blood pressure, maximum heart rate, and more.
